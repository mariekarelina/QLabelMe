## Руководство пользователя QLabelMe

### Введение

**QLabelMe** - это приложение для разметки изображений, предназначенное для
создания и редактирования аннотаций.

Программа позволяет:

- создавать графические фигуры на изображении
- редактировать разметку
- присваивать объектам классы
- сохранять аннотации
- измерять расстояния на изображении

QLabelMe может использоваться для подготовки обучающих данных
для систем компьютерного зрения и машинного обучения.

Поддерживаются следующие типы фигур:

- прямоугольник (Rectangle)
- круг (Circle)
- полилиния (Polyline)
- линия (Line)
- точка (Point)

---

## Интерфейс программы

Главное окно программы состоит из нескольких основных элементов.

### Область изображения

Центральная часть окна отображает текущее изображение и созданную разметку.

В этой области можно:

- создавать фигуры
- редактировать фигуры
- перемещать элементы
- измерять расстояния

### Панель инструментов

Панель содержит инструменты для создания разметки.

Доступные инструменты:

- Rectangle - прямоугольник
- Circle - круг
- Polyline - полилиния
- Line - линия
- Point - точка

### Список изображений

Позволяет переключаться между файлами в выбранной папке.

При выборе изображения программа автоматически
загружает соответствующую разметку.

### Список фигур

Отображает все элементы разметки, созданные на текущем изображении.

Через список фигур можно:

- выбрать фигуру
- удалить фигуру

### Строка состояния

В строке состояния отображается:

- текущий режим работы
- дополнительная информация
- размеры изображения

---

## Создание разметки

Для создания разметки используются инструменты панели.

### Прямоугольник (Rectangle)

Используется для выделения объектов прямоугольной областью.

Порядок работы:

1. Выберите инструмент **Rectangle**
2. Нажмите левую кнопку мыши на изображении
3. Перетащите курсор
4. Отпустите кнопку мыши

После этого будет создан прямоугольник.

---

### Круг (Circle)

Используется для разметки круглых объектов.

Порядок работы:

1. Выберите инструмент **Circle**
2. Укажите центр круга
3. Потяните курсор для задания радиуса

---

### Полилиния (Polyline)

Полилиния используется для создания произвольных многоугольников.

Порядок работы:

1. Выберите инструмент **Polyline**
2. Последовательно ставьте точки на изображении
3. Для завершения построения выполните действие, которое выбрали в настройках

---

### Линия (Line)

Инструмент для создания ломанных линий.

Порядок работы:

1. Выберите инструмент **Line**
2. Последовательно ставьте точки на изображении
3. Для завершения построения выполните действие, которое выбрали в настройках

---

### Точка (Point)

Используется для точечной разметки объектов.

1. Выберите инструмент **Point**
2. Щёлкните на изображении

---

## Редактирование разметки

Созданные фигуры можно редактировать.

Доступные действия:

- перемещение фигуры
- изменение положения узлов
- добавление узлов
- удаление узлов
- изменение класса объекта

### Перемещение фигуры

Чтобы переместить фигуру:

1. Выберите фигуру
2. Перетащите её с помощью мыши

### Редактирование узлов

Узлы фигуры можно перемещать для изменения её формы.

1. Выберите фигуру
2. Перетащите нужный узел

## Настройки

В настройках программы можно изменить параметры отображения.

Доступные параметры:

- толщина линий
- размер узлов
- цвет фигур
- отображение нумерации

Изменения применяются ко всем изображениям.
